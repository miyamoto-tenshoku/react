# 本アプリについて
送信されたチャットログを効果音とともに流し去るアプリです

## 使用したフレームワーク
react

## 使用したライブラリ
- [Framer Motion](https://github.com/framer/motion)
- [React Hotkeys Hook](https://github.com/JohannesKlauss/react-hotkeys-hook)
